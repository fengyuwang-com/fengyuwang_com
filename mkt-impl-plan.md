## 二、每个区块的具体内容

### 1. Hero 区块

**变化**：
- `Science x Art` badge → 替换为思想锚点 `科学与艺术之间`
- h1 `市场学` 保留
- 英雄段落在原来的两段中选择保留核心内容，精简为一到两句

**文案**：

```
思想锚点：科学与艺术之间
h1：市场学
说明：我的市场学理解 —— 从数据出发，用表达抵达。不是单向输出，是双向联系。
```

**与现有内容的对应**：原有 hero 的两段文案的核心内容保留，但精简。未使用的细节移到对应的 section 中。

### 2. Section「数据先说话」

**Section 标题**：数据先说话
**介绍性用语**：分析为本。没有数据支撑的方向，只是猜测。

**内容来源**：
- 原有 `数据先行` pillar card（完整保留）
- 原有"重点关注"第1项：*通过实验、统计和结构化分析去理解消费者需求，而不是靠猜。*
- 原有"科学的一面"段落：*科学的一面，是从数据出发、从市场证明出发，通过实验、统计、行为研究和结果分析去理解人们到底需要什么，产品到底是不是符合需求。*

**展示方式**：标题 + 介绍语 + 卡片/段落

### 3. Section「创意推一把」

**Section 标题**：创意推一把
**介绍性用语**：数据告诉你机会在哪，创意让你抓住它。

**内容来源**：
- 原有`双向联系` pillar card 的后半部分：*公司也要把产品与价值准确传达到消费者*
- 原有"艺术的一面"段落：*海报、图片、视频、声音、文案和视觉语言，都会影响用户如何理解产品、如何记住品牌，以及是否形成情感联系。*
- 原有"重点关注"第2项：*AI、大数据、平台行为和手机使用习惯，正在深刻影响今天的市场学实践。*

**展示方式**：标题 + 介绍语 + 卡片

### 4. Section「听，再说」

**Section 标题**：听，再说
**介绍性用语**：营销不是广播，是对话。先理解，再表达。

**内容来源**：
- 原有 hero-note：*一边是消费者的需求、意图与反馈要被公司真正理解*
- 原有 `双向联系` pillar card 的前半部分：*市场学不是只有公司到消费者的单向输出，它同时也是消费者到公司的反馈回路*
- 原有 `框架与执行并重` pillar card：*4P、STP、服务质量这些框架很重要，但真正关键的是能不能把它们落到执行。*
- 原有"重点关注"第3-5项
- 原有"页面范围"部分内容

**展示方式**：标题 + 介绍语 + 卡片/列表

### 5. Section「全球与本地」

**Section 标题**：全球与本地
**介绍性用语**：语言、地区、文化、市场环境都要一起考虑，全球化不能脱离本地语境。

**内容来源**：
- 原有 `全球化与本地化` pillar card（完整保留）
- 原有"页面范围"中关于多语言、多地区市场的描述

**展示方式**：标题 + 介绍语 + 卡片（可以较短的区块，因为内容量不大）

### 6. Section「做过的事」

**Section 标题**：做过的事
**介绍性用语**：策略写在纸上不值钱，真正验证过才有意义。

**内容来源**：
- 原有 6 张 case card，全部保留，保持 2×3 网格排列
- 可以增加一个引导语提示联系

**展示方式**：标题 + 介绍语 + 2×3 case grid（现有样式）

### 7. 页面底部

**内容**：
- 跨页链接（一行文字，例如：*数字和创意之外，还需要商业判断 → 看看我的投资框架*）
- Bilibili 图标/按钮（如果合适）
- 现有页脚（不动）
