# 网站重新设计计划：以思想为核心的四层呈现架构

## 前提：你的思想是什么（你网站最闪光的东西）

你网站的独特价值不是"一个人会三个领域"，而是 **"一个人用一套统一的底层逻辑，贯穿市场、投资、工程三个领域"**。

这条底层逻辑来自《第一性原理》：

> **赚钱** → 创造价值 → 解决问题 → 识别真需求 → 实践反馈 → 快速迭代 → 验证模型 → 系统复制 → 杠杆

再加上《理性、客观、不坑人》和《行动力+商业脑》揭示的人格底色：

> **"你不是不能赚钱，你是不能骗钱——但你能赚大钱。"**
> **"你不是缺能力，你是缺最小一步落地。"**

以及所有文章共同暗示的认知：

> **"技术会迭代，风口会过期，只有人性永远不变。"**

你的思想可以浓缩为一句话：

> **三个领域 · 一套逻辑 · 干净赚钱 · 长期主义**

## HIG 审计核心发现（578 个问题的根因）

hig-doctor 审计了 13 个文件（6167 个检测点），核心问题分三类：

### 1. 硬编码颜色 — ~300 处
`#ffffff`、`#0f172a`、`#f5f5f7`、`#0071e3` 等写死在 CSS 中，没有使用 CSS 自定义属性（语义 token）。这意味着：
- 暗色模式需要大量 `body[data-theme="dark"]` 覆盖
- 改一个品牌色需要改几十个文件
- 苹果风格的语义化主题无法实现

**改法**：在 `style.css` 顶部定义：

```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f5f5f7;
  --bg-card: #ffffff;
  --text-primary: #1d1d1f;
  --text-secondary: #6e6e73;
  --accent: #0071e3;
  --border: rgba(0,0,0,0.06);
}
[data-theme="dark"] {
  --bg-primary: #0f172a;
  --bg-secondary: #020617;
  --bg-card: #1e293b;
  --text-primary: #f5f5f7;
  --text-secondary: #9fb0c3;
  --accent: #4f7eff;
  --border: rgba(255,255,255,0.08);
}
```

然后用 `var(--bg-primary)` 等替换所有硬编码。

### 2. 固定字号 — ~100 处
`font-size: 16px`、`font-size: 14px` 硬编码。苹果 HIG 要求使用 Dynamic Type / rem 相对单位。`style.css` 中 body 的 `16px` 是好的基准，但子元素应该用 `rem` 而不是 `px`。

**改法**：所有 `font-size: Xpx` → `font-size: Xrem`（除以 16）。

### 3. !important 滥用 — ~50 处
`outline: 0 !important`、`text-shadow: none !important`、`padding: 12px !important` 等。苹果 HIG 把 `!important` 视为 moderate 违规，建议用 specificity 而非 `!important`。

**改法**：`outline: 0 !important` → 只在 `:focus:not(:focus-visible)` 中设置。

### 4. 可访问性缺失
- 缺少 `aria-label`、`aria-current`、`role` 属性
- `<button>` 元素没有明确的文本标签
- 图片缺少 `alt` 文本
- 对比度不足（浅色文字在浅色背景上）

## 一、首页（index.html）改造计划

### 现在的状态
- 轮播 slider（3 张背景图 + 标题 + CTA）
- About 区（头像 + 简介）
- Track cards 区（3 个领域卡片）⭐ 这是做得最好的部分

- 缺少 skip-to-content 导航

### 改造方案：Apple 官网风格

**英雄区（替换轮播）**

```
│         🖼️ 全幅大图（1920x600）              │

│     "市场 · 投资 · 工程"                      │

│     [认识我的思想 →]  [探索能力 →]              │
│                                              │
└──────────────────────────────────────────────┘
```

去掉轮播。一张高质量大图 + 一行大字标题 + 一行副标题 + 两个 CTA。不自动播放，不闪烁。

**What's New 区（新增，替换掉 blog-cards-section）**
```
┌─ What's New ──────────────────────────────┐
│                                           │
│  📰 最新文章                              │
```

- 每张卡片 = kicker（分类）+ 标题 + 发布日期 + 一句话摘要
- "社交动态"自动抓取 LinkedIn 最新帖子

**About 区（保留但简化）**

```diff

+ background: var(--bg-card)
- color: #f8fafc
+ color: var(--text-primary)
```

4. 三条主线（track cards，指向 mkt / invest / portfolio）
